# 个人管理网站

## 项目目标

本项目是一个手机优先、记录负担较低的个人管理网站，用于集中保存身体状态、学习进度和兴趣爱好过程记录。

项目希望把分散在纸质手帐、相册、备忘录和临时想法中的生活信息，整理为可持续记录、可分析、可回顾的个人成长档案。第一版重点验证“结构化记录是否能够产生比纸质手帐更清晰的分析与回顾价值”。

## MVP 范围

第一版围绕“记录 -> 整理 -> 分析 -> 回顾 -> 调整”的基本闭环展开，核心范围包括：

- 身体管理：身体测量记录、运动记录、月经记录、身体状态记录，以及基础趋势与统计。
- 学习管理：学习项目、单次学习记录，以及学习频率、累计投入、项目活跃情况等统计。
- 兴趣爱好：兴趣分类、具体兴趣项目、进展时间线、多图上传、收藏/待打印标记。
- 通用能力：历史记录查看、修改、删除、补记，区分实际发生日期与创建时间。
- 阶段回顾：用图表、统计和文字总结辅助理解近期变化与下一步调整。

第一版明确不做社交、会员、排行榜、复杂积分/等级/成就系统、好友监督、多人协作、医疗诊断或过度确定的健康结论。

## 核心产品原则

- 手机优先：主要记录流程适合移动端快速完成。
- 低负担：减少必填字段，详细说明、照片、感想和备注按需填写。
- 分析有依据：分析必须说明数据基础，数据不足时明确提示无法形成稳定结论。
- 允许中断：不强迫补齐遗漏日期，不因中断记录制造失败感。
- 手帐互补：网站承担结构化记录、分析和素材保存，纸质手帐承担自由表达与最终整理。
- MVP 克制：第一版只实现直接服务于记录、分析、回顾和素材保存的功能。
